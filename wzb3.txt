## 产品核心定位

### 差异化价值主张

**核心问题：** 新手宝妈面临信息过载和真假难辨

**解决方案：**
- ✅ **月龄精准匹配**：自动推送当前月龄的相关内容
- ✅ **Q&A实时互助**：快速获得真实宝妈和医生的回答
- ✅ **内容质量把控**：多维度审核 + 用户评分系统
- ✅ **医生认证体系**：专业人士回答置顶加权
- ✅ **成长记录陪伴**：自动生成宝宝成长报告
- ✅ **创作者激励**：优质内容可获得现金收益

### 目标用户画像

**主要用户（80%）：**
- 孕期宝妈（孕20周-产前）
- 0-3岁宝宝的新手妈妈
- 二胎/三胎宝妈
- 全职妈妈为主，职场妈妈为辅

**次要用户（20%）：**
- 儿科医生、营养师
- 月嫂、育婴师
- 准爸爸、家庭其他成员

## 第一阶段（第1-4周）：基础用户系统

### 1.1 注册登录

**1.1.1 注册方式**
- **手机号注册**（主要）
  - 验证码有效期：5分钟
  - 60秒内只能发送1次
  - 单手机号限注册2个账号
- **微信一键登录**（辅助）
  - 首次登录需绑定手机号
  - 自动获取昵称和头像（可选）

**1.1.2 宝宝信息填写（核心识别）**
```
必填信息：
- 预产期/出生日期
- 宝宝性别（男/女/保密）
- 宝宝昵称

可选信息：
- 出生方式（顺产/剖腹产）
- 喂养方式（纯母乳/混合/纯奶粉）
- 特殊情况（早产/低体重/过敏体质）

多胎支持：
- 最多添加5个宝宝
- 可切换"当前关注的宝宝"
- 自动标记"二胎宝妈""三胎宝妈"
```

**1.1.3 实名认证系统**
- **触发时机**：
  - 首次发布笔记/评论时提示
  - 可在个人中心主动申请
  
- **认证流程**：
  - 上传宝宝出生证明（自动脱敏）
  - 输入姓名 + 身份证号
  - 人工审核（1小时内）
  
- **认证权益**：
  - 显示"已认证" [灰色徽章]
  - 发布权重+10%
  - 降低被误判为营销号风险

### 1.2 身份认证体系

**1.2.1 认证宝妈（蓝V）**
```
申请条件：
- 发布笔记 > 20篇
- 平均点赞 > 50
- 信用分 > 85
- 活跃 > 30天
- 完成实名认证
- 通过社区规范考试（20题，≥85分）

认证权益：
- 蓝V徽章
- 笔记推荐权重+20%
- 评论排序优先
- Q&A回答权重+30%
- 可申请"月龄导师"
- 参与新功能内测

有效期：1年（到期需续期）
```

**1.2.2 医生/专业人士（金V）**
```
适用对象：
- 儿科医生、产科医生
- 儿童营养师
- 儿童心理咨询师
- 康复治疗师

认证材料：
- 执业医师证/专业资格证书
- 工作证/在职证明
- 所在医院/机构名称
- 从业年限证明

认证权益：
- 金色专业徽章（带职称）
- 回答自动置顶、加权
- Q&A优先推送相关问题
- 可设置咨询收费（图文¥9.9-99，电话¥199-999）
- 内容权重最高（+60%）
- 月度专家津贴

认证责任：
- 回答必须基于医学证据
- 不得推荐未经验证的治疗
- 违规处罚：警告 → 取消认证 → 永久封禁

有效期：2年
```

**1.2.3 月嫂/护理师（橙V）**
```
认证条件：
- 持有月嫂证/育婴师证/护理师证
- 从业经验≥3年
- 提供服务案例≥10个

认证权益：
- 橙色徽章
- 护理类回答权重+40%
- 可展示服务经验
- 可发布"服务招募"（需审核）

有效期：2年
```

### 1.3 信用积分系统（简化版）

**1.3.1 初始分数**
- 新用户：100分
- 完成宝宝信息：+5分
- 实名认证：+10分
- 首次发布：+5分

**1.3.2 主要加分规则**
```
内容创作：
- 笔记被浏览1000次：+3分
- 笔记被浏览5000次：+5分
- 笔记被标记S级：+10分
- 笔记被收藏：+2分/次
- 笔记获赞>100：+10分

问答贡献：
- 回答被采纳：+5分/次
- 医生回答被采纳：+10分

社区贡献：
- 有效举报：+3分/次
- 月度无违规：+5分

活跃行为：
- 连续活跃7天：+3分
- 连续活跃30天：+10分
```

**1.3.3 主要扣分规则**
```
违规行为：
- 笔记被删除（轻微）：-10分
- 笔记被删除（严重）：-50分
- 使用软文关键词（未申报）：-5分
- 标记为"营销号"：-30分

恶意行为：
- 买赞刷数据：-50分
- 发布虚假医疗信息：-50分
- 恶意举报：-3分/次

低活跃：
- 连续30天未登录：-5分
- 连续90天未登录：-20分
```

**1.3.4 信用等级**
```
S级（120+）：优秀用户 [钻石徽章] 内容权重+30%
A级（100-119）：优良用户 [金徽章] 内容权重+15%
B级（85-99）：良好用户 [银徽章] 内容权重+5%
C级（70-84）：合格用户 正常权限
D级（50-69）：低信用 内容降权50%，发布需审核
E级（30-49）：高风险 发布限制每天1条，评论需审核
F级（<30）：黑名单预警 发布权冻结7天，必须整改
```

### 1.4 个人中心

**1.4.1 核心数据展示**
```
┌─────────────────────────────────────┐
│ 头像 + 昵称 + 身份标签              │

**1.4.2 功能导航**

## 第二阶段（第5-8周）：内容发布与浏览

### 2.1 笔记发布

**2.1.1 发布入口**
- 底部导航栏"+"按钮
- 点击后选择：发布笔记/发起提问/发布短视频

**2.1.2 编辑器功能**
```
文字编辑：
- 标题（5-50字）
- 正文（10-5000字）
- 支持emoji、换行
- 自动保存草稿（每30秒）

图片上传：
- 最多9张
- 支持JPG/PNG/WEBP（单张≤5MB）
- 支持裁剪、旋转、滤镜
- 自动压缩到1080p
- OCR识别品牌/价格/联系方式

标签系统：
- 月龄标签（自动）：0-1月/1-3月/3-6月等
- 话题标签（必选1-3个）：喂养/睡眠/护理/健康等
- 内容类型（必选1个）：经验分享/求助/晒图/测评/记录

产品推荐申报（强制）：
如果检测到品牌名/logo，必须填写：
- 是否真实使用？
- 使用时长？
- 购买方式？
- 是否获得报酬？
→ 自动添加"包含产品推荐"标签
```

**2.1.3 内容审核**
```
自动审核（即时）：
- 一级违规词 → 直接拦截
- 二级违规词 → 标记"待人工审核"
- 账号风险评分检测

人工审核（2小时内）：
- 审核维度：虚假宣传/软文广告/隐私侵犯/医疗不当
- 审核结果：通过/修改后通过/拒绝
```

### 2.2 短视频模块（新增）

**2.2.1 为什么需要短视频？**
- 新手宝妈更容易用视频表达（不需要写长文）
- 某些场景视频比图文更直观（如"宝宝哄睡技巧"）
- 抖音、快手已培养用户习惯
- **但要控制占比**：短视频占首页内容≤20%

**2.2.2 短视频发布**
```
时长限制：15秒-3分钟

上传方式：
- 从相册选择
- 应用内拍摄

编辑功能：
- 裁剪、封面选择
- 简单滤镜（3-5种）
- 添加文字、贴纸
- 背景音乐（版权音乐库）

审核机制：
- 自动审核：敏感内容检测
- 人工审核：2小时内
```

**2.2.3 短视频浏览**
```
播放方式：
- 沉浸式全屏播放
- 上滑下一个
- 双击点赞

推荐算法：
- 同笔记推荐算法
- 优先推荐同月龄相关视频
```

### 2.3 首页信息流

**2.3.1 推荐算法**
```
推荐分数 = 内容质量(40%) + 月龄匹配(25%) + 话题相关(20%) 
         + 时效性(10%) + 互动热度(5%)

内容质量 = 作者信用分×0.3 + 文字质量×0.3 + 互动率×0.2 + 评论质量×0.2

月龄匹配度：
- 完全匹配：1.0
- 相邻月龄：0.8
- 同阶段：0.6
- 跨阶段：0.3

时效性：
- 24小时内：1.0
- 3天内：0.8
- 7天内：0.6
- >30天：0.2
```

**2.3.2 信息流展示**
```
笔记卡片：
- 作者头像+昵称+身份标签
- 宝宝月龄标签
- 发布时间
- 笔记标题+正文预览
- 图片网格（1-9张）
- 话题标签
- 互动栏：点赞/评论/收藏/分享

特殊标签：
✅ "优质内容" [金色]
✅ "认证医生审核" [绿色]
⚠️ "包含产品推荐" [黄色]
💰 "参与创作者收益计划" [橙色]
```

**2.3.3 筛选与分类**
- 推荐（算法推荐）
- 关注（关注用户内容）
- 同月龄（同月龄宝妈内容）
- 视频（仅短视频）
- 热门（7天高互动）
- 最新（按时间倒序）

**2.4.2 评论功能（重点）**
```

第4优先级：认证宝妈/医生的评论
第5优先级：其他高赞评论

评论审核（参考《评论区防护体系》）：
- 一级违规词（联系方式） → 直接拦截
- 二级违规词（推荐词+品牌） → 标记审核
- 账号风险评分检测
- 链式推广识别

评论可见性：
绿灯（立即显示）：普通问答、真实分享
黄灯（标记显示）：包含推荐但通过审核
红灯（隐藏）：待审核、被举报多次
```

## 第三阶段（第9-12周）：Q&A问答系统

### 3.1 发起提问

**3.1.1 提问编辑器**
```
问题标题（10-100字）
详细描述（20-2000字，支持图片）

选择分类（必选）：
- 喂养/睡眠/护理/健康/发育/产后/其他

标记紧急程度：
- 普通（正常优先级）
- 急需（优先显示和推送，每月限3个）

选择接受的回答者：
- 所有人（默认）
- 认证宝妈
- 医生/专业人士
- 我关注的用户

宝宝信息自动填充：
"我的宝宝：3月龄/男宝"（可修改）
```

**3.1.2 提问前检查**
```
相似问题推荐：
- 输入标题时实时搜索已有问题
- 如存在相似问题，提示："以下问题可能对你有帮助"
- 用户可选择"查看这些问题"或"继续提问"

问题质量检测：
- 标题过短 → 提示补充
- 包含联系方式 → 拦截
```

### 3.2 回答机制

**3.2.1 回答编辑器**
```
文字输入（50-2000字）
支持上传图片（最多5张）
支持插入链接（限平台内链接）

认证标识：
- 医生/认证宝妈回答自动显示身份标签
- 回答发布后带"认证回答"标记
```

**3.2.2 回答排序**
```
优先级排序：
1. 提问者标记的"最佳答案"（置顶，绿色背景）
2. 医生/专业人士的回答（金色边框）
3. 认证宝妈的回答（蓝色边框）
4. 提问者点赞的回答
5. 高赞回答（点赞>10）
6. 最新回答
```

**3.2.3 最佳答案机制**
```
标记最佳答案（仅提问者可标记）：
- 每个问题只能标记1个
- 标记后自动置顶
- 显示"最佳答案" [绿色徽章]

标记后奖励：
- 回答者：信用分+5，采纳率+1
- 提问者：问题状态变"已解决"

取消最佳答案：
- 提问者可取消
- 取消需填写原因
- 回答者扣除对应奖励
```
