# 宝妈说 - 完整功能清单（细分版）

> **开发周期：32周（8个月）**  
> **核心原则：先做核心交互，再做内容质量，最后做社交强化**

---

## 第一阶段（第1-4周）：基础用户与账号体系

### 1.1 注册登录模块

#### 1.1.1 注册流程
- **手机号注册**
  - 输入手机号 → 获取验证码 → 输入验证码 → 验证成功
  - 验证码有效期：5分钟
  - 验证码发送限制：60秒内只能发送1次
  - 同一手机号24小时内注册失败超过5次 → 24小时冻结
  - 单手机号限注册2个账号（防批量注册）

- **微信一键登录**（可选）
  - 调用微信SDK授权登录
  - 自动获取微信昵称和头像（可选择是否使用）
  - 首次登录需要绑定手机号

- **用户名与密码设置**
  - 用户名长度：3-15字符
  - 支持中文、英文、数字、下划线
  - 用户名不能重复（实时校验）
  - 用户名禁止包含营销词汇：
    - "微商""代购""商城""货源""批发"
    - "客服""官方""旗舰店"
  - 密码强度要求：≥8字符，必须包含字母+数字
  - 支持"忘记密码"通过验证码重置

- **隐私协议与用户条款**
  - 首次注册必须勾选同意
  - 包含：
    - 《用户协议》
    - 《隐私政策》
    - 《社区内容规范》v1.0
  - 可展开查看详细内容

#### 1.1.2 宝宝信息填写（核心识别）
- **首次注册时触发**
  - 引导文案："告诉我们你的宝宝信息，帮你找到最适合的内容"

- **选项1：输入预产期（孕期宝妈）**
  - 日期选择器（精确到日）
  - 自动计算"预计还有X天/X周"
  - 系统自动标记为"孕期宝妈"
  - 预产期到达后自动变为"待产"状态
  - 用户可手动更新为"已出生"

- **选项2：输入出生日期（已出生）**
  - 日期选择器（精确到日）
  - 自动计算宝宝当前月龄（精确到天）
  - 月龄自动更新（每天0点刷新）
  - 显示宝宝发育阶段标签：
    - 0-1月龄：新生儿期
    - 1-3月龄：婴儿早期
    - 3-6月龄：婴儿中期
    - 6-12月龄：婴儿晚期
    - 1-2岁：幼儿早期
    - 2-3岁：幼儿中期
    - 3岁+：学龄前

- **选项3：我有多个宝宝（二胎/多胎）**
  - 支持添加最多5个宝宝信息
  - 每个宝宝独立填写出生日期
  - 选择"当前关注的宝宝"（默认显示最小的宝宝）
  - 可在设置中随时切换关注对象
  - 系统自动标记为"二胎宝妈""三胎宝妈"

- **宝宝信息字段**
  - 宝宝姓名/昵称（必填，20字以内）
  - 出生/预产日期（必填）
  - 性别（必选：男/女/保密）
  - 出生方式（可选：顺产/剖腹产/其他）
  - 喂养方式（可选：纯母乳/混合/纯奶粉/未决定）
  - 是否有特殊情况（可选：早产/低体重/过敏体质等）

- **宝宝信息修改**
  - 用户可随时在"个人中心-宝宝信息"中修改
  - 修改后月龄标签自动更新
  - 系统记录修改历史（防止虚假信息/频繁切换身份）
  - 修改频率限制：月龄改动30天内只能改1次

#### 1.1.3 实名认证系统
- **普通宝妈实名认证**（非强制）
  - 触发时机：
    - 第一次发布笔记时提示认证
    - 第一次发布评论时提示认证
    - 在个人中心主动申请
  
  - 认证流程：
    - 上传宝宝出生证明照片（系统自动脱敏处理）
    - 输入真实姓名 + 身份证号
    - 人工审核（1小时内）
  
  - 认证后权益：
    - 个人资料显示"已认证" [灰色徽章]
    - 发布内容权重+10%
    - 评论排序优先级提升
    - 降低被误判为营销号的风险
  
  - 认证信息保护：
    - 实名信息仅平台可见，不对外展示
    - 加密存储，符合《个人信息保护法》

#### 1.1.4 身份认证系统

**1. 认证宝妈（蓝V认证）**
- **申请条件**
  - 发布笔记>20篇
  - 平均点赞>50
  - 历史内容零违规（信用分>85）
  - 平台活跃>30天
  - 完成实名认证
  
- **认证流程**
  - 用户在个人中心点击"申请认证宝妈"
  - 系统自动检测是否满足条件
  - 需要通过"社区规范考试"（20道选择题，考察对社区规则的理解）
  - 考试通过分数线：≥85分
  - 平台人工审核（1-3工作日）
  
- **认证后权益**
  - 获得蓝V标签 [蓝色徽章]
  - 笔记自动获得初始推荐权重+20%
  - 评论排序优先级提升
  - 回答Q&A时权重提升
  - 可申请成为"月龄导师"
  - 参与新功能内测
  
- **认证维护**
  - 认证有效期：1年
  - 到期后重新审核（简化流程）
  - 违规3次取消认证

**2. 医生/专业人士认证（金色徽章）**
- **适用对象**
  - 儿科医生
  - 产科医生
  - 营养师
  - 儿童心理咨询师
  - 康复治疗师
  
- **认证材料**
  - 医学证书照片（执业医师证/专业证书）
  - 工作证/从业证件
  - 所在机构/医院名称
  - 科室/职位证明
  - 个人简介（工作年限、专业领域）
  
- **认证流程**
  - 用户在个人中心点击"专业认证"
  - 上传认证材料
  - 平台人工审核（1-3工作日）
  - 必要时电话核实
  
- **认证后权益**
  - 获得"医生" [金色徽章] 或"营养师" [绿色徽章]
  - 医生回答自动置顶、加权
  - Q&A模块中回答优先推荐
  - 可设置咨询收费（可选）
  - 内容权重最高
  
- **认证责任**
  - 医生内容违规（虚假宣传、误导性建议）→ 直接永久封禁
  - 信任度要求更高，审核更严格
  - 每季度抽查认证有效性

**3. 月嫂/护理师认证（橙色徽章）**
- **认证条件**
  - 相关工作经验≥3年
  - 持有资格证书（月嫂证/育婴师证/护理师证）
  - 无不良从业记录
  
- **认证流程**
  - 上传资格证书照片
  - 上传工作经历证明
  - 平台人工审核（3-5工作日）
  
- **认证后权益**
  - 获得"月嫂" [橙色徽章]
  - 护理相关领域回答有权重提升
  - 可在个人简介中展示服务经验
  
- **认证维护**
  - 认证有效期：2年
  - 到期重新提交材料审核

#### 1.1.5 账号风险评分系统（防营销号核心）

**风险评分模型**
```
风险评分 = 基础分(100)
         - 账号年龄(<3天 -30分，<7天 -20分)
         - 无头像(-20分)
         - 无宝宝信息(-30分)
         - 未实名认证(-10分)
         - 昵称含营销词(-40分)
         - 注册后未发笔记只评论(-50分)
         - 个人简介含联系方式(-50分)
         - 发布频率异常(每天>5条 -20分)
         - 评论频率异常(每天>50条 -30分)
         - 被举报次数(每次 -5分)
         - 内容被删除次数(每次 -10分)
```

**风险等级划分**
- **低风险（80-100分）**
  - 内容正常发布，不需要审核
  - 评论立即可见
  
- **中风险（50-79分）**
  - 发布内容需要人工审核（2小时内）
  - 评论需要审核后可见
  - 系统提示："完善账号信息可提升发布权限"
  
- **高风险（<50分）**
  - 发布权限冻结，需要完成实名认证
  - 评论不可见（仅自己可见）
  - 系统提示："你的账号存在异常行为，请联系客服"

**风险评分更新**
- 评分每24小时自动更新一次
- 评分对用户不可见（仅后台监控）
- 评分日志记录所有变化

**风险等级处理**
- 高风险账号会收到站内信通知
- 提供"账号申诉"入口
- 申诉通过后解除限制

#### 1.1.6 账号信用积分系统

**初始分数**：100分

**加分规则**
- 发布笔记被浏览>100次：+1分
- 发布笔记被浏览>1000次：+3分
- 笔记被收藏：+2分
- 笔记获赞>10：+3分
- 笔记获赞>100：+10分
- 回答被采纳：+5分
- 被认证为"认证宝妈"：+50分
- 月度无违规：+5分
- 有效举报数>5：+3分
- 完成实名认证：+10分
- 连续活跃30天：+5分

**扣分规则**
- 被举报1次：-2分
- 举报被核实（内容违规）：-5分
- 笔记被删除：-10分
- 使用软文关键词：-5分
- 账号被标记为"营销号"：-30分
- 评论被删除（垃圾信息）：-1分
- 参与"评论小分队"：-20分
- 恶意举报（举报不实）：-3分
- 发布虚假医疗信息：-50分

**分数后果**
- **>85分：优质用户**
  - 内容优先推荐
  - 有社群特权
  - 个人资料显示"优质创作者" [星星标签]
  
- **70-85分：正常用户**
  - 正常待遇
  
- **50-70分：低信用用户**
  - 内容降权
  - 需要人工审核后发布
  - 收到系统提示："你的信用分较低，请遵守社区规范"
  
- **30-50分：高风险用户**
  - 发布限制（每天最多1条）
  - 收到警告通知
  - 账号页面显示"该账号信用分较低"
  
- **<30分：黑名单预警**
  - 发布权冻结7天
  - 进入黑名单预警名单
  - 必须完成整改（删除违规内容、申诉说明）才能恢复

**信用分展示**
- 在个人中心可查看自己的信用分
- 信用分变化有详细记录
- 可查看加分/扣分原因

### 1.2 个人档案系统

#### 1.2.1 个人信息展示
- **头像**
  - 支持上传格式：JPEG/PNG/WEBP
  - 文件大小限制：最大2MB
  - 系统自动压缩到200x200px
  - 支持裁剪预览（1:1比例）
  - 支持默认头像（生成昵称首字母头像，5种颜色随机）
  - 头像审核（防止不当头像）：
    - 自动识别：敏感内容、广告、二维码
    - 人工审核（疑似问题头像）
    - 违规头像自动替换为默认头像

- **昵称**
  - 长度：3-15字符
  - 支持中文、英文、数字、特殊符号（限定范围）
  - 昵称唯一性检测（实时校验）
  - 昵称禁止包含：
    - 营销词汇："微商""代购""批发"等
    - 联系方式：电话、微信号、QQ号
    - 不雅词汇（敏感词库）
  - 修改限制：7天内最多改1次（防止账号切换身份）
  - 修改记录可查询（防止恶意更改）

- **简介**
  - 长度：0-100字（可选字段）
  - 支持标签选择：
    - "新手宝妈""二胎宝妈""三胎宝妈"
    - "孕期宝妈""职场妈妈""全职妈妈"
    - "育儿博主""经验分享者"
  - 简介敏感词过滤（自动拦截联系方式）
  - 支持添加地区：如"北京 海淀"

- **宝宝展示**
  - 显示当前关注的宝宝信息（不暴露真名）
  - 显示月龄标签（如"3月龄宝妈" [badge]）
  - 显示宝宝性别icon（男宝：♂️ / 女宝：♀️）
  - 显示多宝家庭标签（"二胎宝妈""三胎宝妈"）
  - 可切换显示的宝宝（二胎/多胎家庭）

- **身份标签展示**（按优先级排列）
  - 医生 [金色徽章]
  - 认证宝妈 [蓝V]
  - 月嫂/护理师 [橙色徽章]
  - 社区守护者 [星星徽章]
  - 月度活跃用户 [火焰徽章]
  - 优质创作者 [星星标签]

#### 1.2.2 个人成就系统

**徽章系统**（共20个徽章）
- **新手系列**
  - 新手上路：首次发布笔记
  - 初次互动：首次评论
  - 社交达人：首次关注他人
  
- **内容创作系列**
  - 内容创作者：发布笔记>10篇
  - 勤奋创作者：发布笔记>50篇
  - 创作大师：发布笔记>100篇
  - 话题达人：在某话题下笔记>5篇
  - 月度优质创作者：月度笔记被赞>100次
  
- **社区贡献系列**
  - 社区守护者：有效举报>10次
  - 知识贡献者：回答被采纳>10次
  - 最佳回答者：单月回答被采纳>5次
  - 互动达人：月度评论>50条（非垃圾评论）
  
- **活跃系列**
  - 月度活跃：单月活跃天数>20天
  - 连续签到：连续签到7天/30天/100天
  
- **特殊成就**
  - 百赞笔记：单篇笔记获赞>100
  - 千赞笔记：单篇笔记获赞>1000
  - 人气宝妈：粉丝数>100
  - 超人气宝妈：粉丝数>1000

**等级系统**（5个等级）
- **Lv1：新手宝妈**
  - 条件：注册新用户
  - 权益：基础功能
  
- **Lv2：活跃宝妈**
  - 条件：发布笔记>5篇
  - 权益：可参与话题投票
  
- **Lv3：资深宝妈**
  - 条件：发布笔记>20篇 + 总点赞>100
  - 权益：可申请月龄社群管理员
  
- **Lv4：认证宝妈**
  - 条件：发布笔记>50篇 + 总点赞>500 + 蓝V认证
  - 权益：优先推荐、社群特权
  
- **Lv5：殿堂级宝妈**
  - 条件：发布笔记>100篇 + 总点赞>1000 + 粉丝>500
  - 权益：最高推荐权重、可申请官方合作

**排行榜系统**
- **月度排行**
  - 月度最佳回答者（回答被采纳数）
  - 月度最活跃宝妈（活跃天数）
  - 月度人气宝妈（粉丝增长数）
  
- **周度排行**
  - 周度优质内容（笔记热度）
  - 周度热心宝妈（回答数）
  
- **总榜**
  - 总粉丝榜
  - 总点赞榜
  - 总创作榜

**排行榜展示**
- 显示Top 10
- 用户可查看自己的排名
- 每周一/月初更新

#### 1.2.4 内容管理

**我的笔记**
- **列表展示**
  - 显示所有发布的笔记（按时间倒序）
  - 笔记卡片显示：
    - 封面图（第一张图片或默认图）
    - 标题
    - 发布时间
    - 浏览数、评论数、点赞数、收藏数
    - 笔记状态：草稿/审核中/已发布/已下架
  
- **操作菜单**（右滑或长按显示）
  - 编辑（仅草稿和审核中状态可编辑）
  - 删除（二次确认）
  - 隐藏（对外不可见，但不删除）
  - 分享（生成分享链接）
  - 数据统计（查看详细数据）
  
- **批量操作**
  - 支持多选
  - 批量删除
  - 批量设置评论权限

**我的回答**
- **列表展示**
  - 显示所有回答过的问题
  - 回答卡片显示：
    - 问题标题
    - 回答预览（前50字）
    - 采纳状态：已采纳/未采纳
    - 点赞数、回复数
    - 回答时间
  
- **筛选功能**
  - 全部回答
  - 已采纳
  - 未采纳
  - 医生回答（仅认证医生）
  
- **采纳率展示**

2.2 笔记浏览模块
2.2.1 首页信息流
默认排序算法（智能推荐）
推荐分数 = 内容质量分(40%)
         + 月龄匹配度(25%)
         + 话题相关度(20%)
         + 时效性(10%)
         + 互动热度(5%)

内容质量分 = 作者信用分 × 0.3
           + 文字质量(长度/细节) × 0.3
           + 互动率(点赞/浏览) × 0.2
           + 评论质量 × 0.2

月龄匹配度：
- 完全匹配（同月龄）：1.0
- 相邻月龄（±1月）：0.8
- 同阶段（0-3月/3-6月等）：0.6
- 跨阶段：0.3

话题相关度：
- 基于用户历史浏览话题
- 常浏览的话题权重更高
- 计算公式：用户浏览该话题次数 / 总浏览次数

时效性：
- 24小时内：1.0
- 3天内：0.8
- 7天内：0.6
- 30天内：0.4
- >30天：0.2

互动热度：
- (点赞数 × 1 + 评论数 × 2 + 收藏数 × 3) / 发布天数

回答入口

  - 低风险账号：立即发布
  - 中风险账号：标记为"待审核"（2小时内）
  - 高风险账号：拦截，提示完善账号信息
  - 医疗敏感内容：必须人工审核

#### 回答展示与排序
- **排序规则**
  ```
  优先级排序：
  1. 提问者标记的"最佳答案"（置顶，绿色背景）
  2. 医生/专业人士的回答（金色边框）
  3. 认证宝妈的回答（蓝色边框）
  4. 提问者点赞的回答
  5. 高赞回答（点赞数>10）
  6. 最新回答
  ```

  - 回答者头像 + 昵称 + 身份标签
  - 宝宝月龄标签
  - 回答内容（支持折叠，默认显示前200字）
  - 回答时间
  - 点赞数、评论数
  - 操作按钮：点赞、评论、举报、采纳（仅提问者可见）

#### 最佳答案机制
- **标记最佳答案**
  - 仅提问者可标记
  - 每个问题只能标记1个最佳答案
  - 标记后自动置顶
  - 显示"最佳答案" [绿色徽章]
  
- **标记后奖励**
  - 被采纳的回答者：
    - 信用分+5
    - 收到通知："你的回答被采纳为最佳答案"
    - 采纳率统计+1
  - 提问者：
    - 问题状态变为"已解决"
    - 可继续接受其他回答
  
- **取消最佳答案**
  - 提问者可取消标记
  - 取消后回答者扣除对应奖励
  - 需要填写取消原因

#### 回答互动
- **点赞回答**
  - 机制同笔记点赞
  - 高赞回答排序提升
  
- **评论回答**

  - 提问者评论自动显示"提问者" [标签]

    - 广告/推广

#### 问答推荐算法
```
推荐分数 = 问题质量(30%)
         + 紧急程度(20%)
         + 月龄匹配度(25%)
         + 时效性(15%)
         + 回答活跃度(10%)

问题质量 = 标题长度 × 0.3
         + 描述详细度 × 0.4
         + 提问者信用分 × 0.3

时效性：
- 24小时内：1.0
- 3天内：0.8
- 7天内：0.6
- 14天内：0.4
- >14天：0.2

回答活跃度：
- 无回答：1.0（优先显示）
- 1-2个回答：0.8
- 3-5个回答：0.6
- >5个回答：0.4
- 已标记最佳答案：0.2（降低优先级）
```

## 第五阶段（第17-20周）：内容组织与指南

### 5.1 阶段指南系统

#### 5.1.1 智能指南推送
- **触发机制**
  - 用户首次登录时推送当前月龄指南
  - 宝宝月龄变化时自动推送新阶段指南
  - 用户可在"成长指南"模块主动查看

- **指南内容结构**
  ```
  每个月龄指南包含：
  1. 本月龄宝宝发育特点
  2. 喂养建议（母乳/奶粉/辅食）
  3. 睡眠规律与建议
  4. 发育里程碑（大动作/精细动作/语言/认知）
  5. 常见问题与解决方案
  6. 疫苗接种提醒
  7. 体检项目提醒
  8. 本月龄注意事项
  ```

- **指南来源**
  - 平台精选笔记（高赞、认证用户）
  - 医生/专业人士审核内容
  - 权威医学资料整理
  - 用户真实经验汇总

#### 5.1.2 新手避坑手册
- **手册结构**
  - 每个月龄对应一份"避坑手册"
  - 列出该月龄最常见的5个误区
  - 每个误区包含：
    - 误区描述
    - 为什么是误区（科学解释）
    - 正确做法
    - 真实案例
    - 参考链接

- **误区示例**
  ```
  0-1月龄：
  ❌ 误区1："新生儿要睡硬枕头定型"
  ✓ 正确：新生儿不需要枕头，睡枕头可能导致窒息风险
  
  ❌ 误区2："满月后可以竖抱"
  ✓ 正确：满月宝宝脊椎尚未发育完全，应继续横抱
  
  3-6月龄：
  ❌ 误区1："4个月一定要加辅食"
  ✓ 正确：辅食添加应看宝宝发育信号，不必严格按月龄
  
  ❌ 误区2："米粉要加在奶里喂"
  ✓ 正确：辅食应单独喂，不要混在奶里
  ```

- **手册更新**
  - 每季度更新一次
  - 根据用户反馈补充新误区
  - 医生审核后发布

#### 5.1.3 成长记录单
- **数据记录项**
  ```
  基础数据：
  - 体重（kg）
  - 身长/身高（cm）
  - 头围（cm）
  - 胸围（cm）
  
  日常数据：
  - 每日睡眠时长
  - 每日喂养量/次数
  - 每日排便次数
  - 体温
  
  发育里程碑：
  - 第一次笑
  - 第一次翻身
  - 第一次坐起
  - 第一次爬行
  - 第一次站立
  - 第一次走路
  - 第一次说话
  - 第一次长牙
  ```

- **数据录入方式**
  - 手动录入：用户在"成长记录"页面填写
  - 打卡自动录入：从打卡数据自动提取
  - 语音录入：语音输入数据（AI识别）
  - 批量导入：从体检报告自动识别（OCR）

- **数据可视化**
  - 生长曲线图（体重/身高/头围）
  - 对标国标发育曲线（WHO/中国儿童发育标准）
  - 显示百分位数（如"体重处于50%分位"）
  - 趋势分析："宝宝生长速度正常/偏快/偏慢"

- **异常数据提醒**
  - 体重增长过快/过慢 → 提醒："建议咨询儿科医生"
  - 身高低于3%分位 → 提醒："建议进行生长激素检查"
  - 头围异常 → 提醒："建议进行头颅B超检查"
  - 发育里程碑延迟 → 提醒："建议进行发育评估"

- **报告生成**
  - 月度成长报告：汇总本月数据
  - 年度成长报告：回顾全年成长历程
  - 可下载为PDF
  - 可分享到社交平台

#### 5.2.1 知识分类
